# translation-web
openai을 이용한 번역 사이트

## 필요 조건


```
pip install openai
pip install flask
pip install PyPDF2~=2.0
pip install python-dotenv

```

### 실행 방법

```
git clone https://github.com/Lee-Siyoung/eng-kor-translation-web
cd eng-kor-translation-web
flask run
```

http://127.0.0.1:5000/  실행
