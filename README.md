# eng-kor-translation-web
영어 - 한국어 번역 사이트

## 필요 조건

openai
```
pip install openai
```

flask
```
pip install flask
```

### 실행 방법

```
git clone https://github.com/Lee-Siyoung/eng-kor-translation-web
cd eng-kor-translation-web
flask run
```
http://127.0.0.1:5000/  실행
